## 1. Autopilot vision + change plan (COMPLETE — proposal delivered)

- docs/autopilot-vision.md NEW: the full design.
  - Honest inventory: 9 of 11 lifecycle steps are manual today; only injection and
    guard are automatic — exactly the two a user would keep.
  - Target spec: ask once, act always, show weekly, undo anytime.
  - Core design move: a MACHINE CURATOR replaces the human queue instead of deleting
    curation — existing gates + reviewer screen (reused from adopt, fail-closed) +
    dry-run canary gate with batch rollback + probation confidence + auto-retire
    sweep + git audit trail + one-click undo.
  - `raph pulse` = the only new verb: SessionEnd hook -> detached child -> mine ->
    budget/E-LIMIT check -> distill -> curate -> atlas refresh -> compile -> one
    pulse event. Every step fails open. No daemon, no scheduler, event-driven only.
  - Global consent (`consent: all` + ignore list) + `mode: autopilot` + dial FULL.
  - Onboarding happens IN the chat: first-SessionStart envelope has the agent ask
    the one consent question, then `raph arise --autopilot`. Never asks again.
  - Passive visibility: ≤150-token weekly digest (7-day throttle, silent when
    empty, security lessons always highlighted); console becomes the read-mostly
    window with one-click undo.
  - Security lessons AUTO-APPROVED per the owner's explicit call — recorded as a
    REVERSAL of §11.11 (to be written as ARCHITECTURE §11.13 when 17.1 builds);
    residual risk stated plainly in the doc (advisory-only + audit + undo bound it).
  - ONE floor kept: quarantined (injection-suspect) content never machine-activates
    — but it never nags either (silent, digest-counted, 30-day tombstone). Rare
    (2 items ever). Reasoning: no machine screen can clear content whose defining
    property is that it tried to manipulate the machine.
- TASKS.md: Phase 17 "Autopilot" added as PROPOSED, 6 milestones in dependency
  order (consent/mode -> machine curator BEFORE the unattended loop -> pulse+hook ->
  atlas-in-pulse -> onboarding+digest -> default flip + v0.2.0). ~6 sessions.
